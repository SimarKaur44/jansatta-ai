# Jansatta AI 🗺️
### 20 million voices. One map. Zero excuses.

**Team EchoMappers | India Innovates 2026 | IILM University**

Jansatta AI turns the unheard voices of citizens into a real-time map of urgent governance need.

## The Problem
- 4,000+ urban wards across India with NO real-time citizen feedback loop
- 90% of civic conversations happen in regional languages — completely untracked
- 47 days average between a citizen complaint and government awareness

## What It Does
- Ingests YouTube comments + regional news feeds in Hindi, Hinglish, Marathi, Bengali, Tamil
- Runs IndicBERT + MuRIL NLP to detect language, sentiment, and issue category
- Maps complaints to Delhi ward boundaries via GeoJSON
- Generates Priority Alerts for governance action
